Enhanced Artifacts

All three enhancements build on a single artifact — the CS 360 Weight Tracker Android app. Each enhancement builds on the previous one rather than on the original, so the original source is preserved unchanged as a baseline in every repository.

Enhancement One — Software Design and Engineering

Refactored the app for maintainability and security:

  • Created InputValidator.java to centralize all field validation
  • Created PasswordUtils.java implementing salted SHA-256 password hashing
  • Added try-finally cursor management throughout
  • Replaced anonymous logic with named, testable methods

Added analytics capabilities to the app:

Enhancement Three — Databases

Hardened the data layer:

  • Added DatabaseManager.java implementing the Repository pattern
  • Wrapped all CRUD operations in transactions with audit logging and input sanitization
  • Added CSV export and date-range filtering
  • Upgraded schema to version 3 with composite indexes, enforced foreign keys, ON DELETE CASCADE, and a non-destructive migration path
